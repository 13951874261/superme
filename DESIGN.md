# Design

## Theme
A dark-mode execution panel paired with a clean, high-contrast light-mode content workspace. 

## Colors
- **Primary Accent**: #FF5722 (Orange-Red) for execution controls, primary call-to-actions, and highlights.
- **Secondary Accent**: #2563eb (Blue-600) for material tabs and selection modes.
- **Backgrounds**: 
  - Slate-50 / White for content card wrappers.
  - #202124 / #121212 for dark execution panels and logs.
- **Ink / Text**:
  - Light-mode: #202124 for headings, #4a5568 for body text.
  - Dark-mode: #ffffff for titles, #a0aec0 for logs and secondary texts.

## Typography
- **Headings**: Tracked, uppercase, bold sans-serif kicked brows for steps and labels.
- **Body**: System-default UI sans-serif at 12px-14px with adequate line-height (1.5) for readability.
- **Monospace**: Consolas / SFMono-Regular for system logs and code references.
